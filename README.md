# My Finance Application

My Finance is a web application for organizing personal finances.

## Technologies/frameworks used

## Front:
- React (js)
- React-Toastify
- PrimeReact
- Bootstrap
- React Router Dom
- JWT Security

## Back:
- Springboot (java)
- Project Lombok
- Mockito JUnit (used in unit tests)
- H2 database
- PostgreSQL
- JWT Security
